# 登录与注册接口文档

## 通用接口规范

### 1) 统一返回结构

所有接口统一返回：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

- `code`：业务状态码，`200` 表示成功
- `message`：提示信息
- `data`：响应数据，列表接口通常返回分页对象，删除等操作可为 `null`

### 2) 错误码约定

| 错误码 | 含义 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或登录已过期 |
| 403 | 无权限访问 |
| 404 | 资源不存在 |
| 409 | 业务冲突（如状态不允许、数据重复） |
| 500 | 服务器内部错误 |

### 3) 分页参数约定

- 请求参数统一使用：`page`（页码，从 1 开始）、`pageSize`（每页条数）
- 分页响应统一包含：`records`、`total`、`page`、`pageSize`

### 4) 时间格式约定

- 日期：`yyyy-MM-dd`
- 日期时间：`yyyy-MM-dd HH:mm:ss`
- 默认时区：`Asia/Shanghai`（东八区）

### 5) 鉴权方式约定

- 登录、注册、邀请码注册等公开接口无需 Token
- 其余受保护接口需携带 JWT 访问令牌
- 请求头统一使用：`Authorization: Bearer <token>`

## 一、登录

### 1.1 用户登录

**接口路径：** `POST /api/auth/login`

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| username | String | ✅ | 用户名 |
| password | String | ✅ | 密码（MD5加密传输，即 32位小写十六进制） |

**请求示例：**

```json
{
  "username": "admin",
  "password": "e10adc3949ba59abbe56e057f20f883e"
}
```

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码，200表示成功 |
| message | String | 提示信息 |
| data | Object | 登录结果数据 |

**data 字段说明：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | String | JWT访问令牌 |
| userId | Long | 用户ID |
| username | String | 用户名 |
| tenantId | Long | 租户ID（平台管理员为null） |
| tenantName | String | 租户名称（平台管理员为null） |
| roleCode | String | 角色编码 |
| roleName | String | 角色名称 |
| shopId | Long | 门店ID（店长角色才有值） |
| shopName | String | 门店名称（店长角色才有值） |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 1,
    "username": "admin",
    "tenantId": null,
    "tenantName": null,
    "roleCode": "super_admin",
    "roleName": "平台管理员",
    "shopId": null,
    "shopName": null
  }
}
```

**店长登录成功示例：**

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 10,
    "username": "zhangsan",
    "tenantId": 5,
    "tenantName": "某某餐饮品牌",
    "roleCode": "shop_owner",
    "roleName": "门店店长",
    "shopId": 100,
    "shopName": "XX门店(朝阳店)"
  }
}
```

**失败响应示例：**

```json
{
  "code": 401,
  "message": "用户名或密码错误",
  "data": null
}
```

## 二、注册

### 2.1 租户用户注册

适用于：新用户自助注册（自动创建租户和租户管理员账号）

**接口路径：** `POST /api/auth/register`

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| username | String | ✅ | 用户名 |
| password | String | ✅ | 密码（MD5加密传输） |

**请求示例：**

```json
{
  "username": "zhangsan",
  "password": "e10adc3949ba59abbe56e057f20f883e"
}
```

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息 |
| data | Object | 创建的用户信息 |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "用户创建成功",
  "data": {
    "id": 15,
    "username": "zhangsan",
    "tenantId": 5,
    "roleCode": "tenant_admin"
  }
}
```

**失败响应示例（用户名已存在）：**

```json
{
  "code": 409,
  "message": "该用户名已存在",
  "data": null
}
```

---

### 2.2 店长自主注册

适用于：门店店长通过邀请码自助注册

**接口路径：** `POST /api/auth/register/shop-manager`

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| username | String | ✅ | 用户名 |
| password | String | ✅ | 密码 |
| inviteCode | String | ✅ | 邀请码（需验证有效性） |

**请求示例：**

```json
{
  "username": "zhangsan",
  "password": "e10adc3949ba59abbe56e057f20f883e",
  "inviteCode": "SHOP2026ABC123"
}
```

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息 |
| data | Object | 注册结果（包含登录信息） |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 16,
    "username": "zhangsan",
    "tenantId": 5,
    "tenantName": "某某餐饮品牌",
    "roleCode": "shop_owner",
    "roleName": "门店店长",
    "shopId": 101,
    "shopName": "XX门店(海淀店)"
  }
}
```

**失败响应示例（邀请码无效）：**

```json
{
  "code": 400,
  "message": "邀请码无效或已过期",
  "data": null
}
```

## 三、密码相关

### 3.1 重置用户密码

适用于：管理员重置用户密码

**接口路径：** `PUT /api/auth/password/reset/{userId}`

**路径参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| userId | Long | ✅ | 用户ID |

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息（包含新密码） |
| data | null | 无数据 |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "密码重置成功，新密码为：Xyz789012",
  "data": null
}
```

---

### 3.2 修改密码

适用于：用户修改自己的密码

**接口路径：** `PUT /api/auth/password/change`

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| oldPassword | String | ✅ | 旧密码（MD5加密传输） |
| newPassword | String | ✅ | 新密码（MD5加密传输） |

**请求示例：**

```json
{
  "oldPassword": "e10adc3949ba59abbe56e057f20f883e",
  "newPassword": "c33367701511b4f6020ec61ded352059"
}
```

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息 |
| data | null | 无数据 |

**失败响应示例（旧密码错误）：**

```json
{
  "code": 400,
  "message": "旧密码错误",
  "data": null
}
```

## 四、邀请码管理

### 4.1 生成店长邀请码

适用于：租户管理员为门店生成店长注册邀请码

**接口路径：** `POST /api/invite-code/generate`

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| shopId | Long | ✅ | 门店ID |

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息 |
| data | Object | 邀请码信息 |

**data 字段说明：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| inviteCode | String | 邀请码 |
| shopId | Long | 门店ID |
| shopName | String | 门店名称 |
| expireTime | String | 过期时间 |
| status | String | 状态（unused / used / expired） |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "邀请码生成成功",
  "data": {
    "inviteCode": "SHOP2026ABC123",
    "shopId": 101,
    "shopName": "XX门店(海淀店)",
    "expireTime": "2026-05-17 23:59:59",
    "status": "unused"
  }
}
```

---

### 4.2 查询邀请码列表

适用于：租户管理员查看本店品牌的邀请码

**接口路径：** `GET /api/invite-code/list`

**查询参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| shopId | Long | ❌ | 门店ID（筛选） |
| status | String | ❌ | 状态（unused / used / expired） |
| page | Integer | ❌ | 页码，默认1 |
| pageSize | Integer | ❌ | 每页条数，默认10 |

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息 |
| data | Object | 分页数据 |

**data 字段说明：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| records | Array | 邀请码列表 |
| total | Long | 总条数 |
| page | Integer | 当前页 |
| pageSize | Integer | 每页条数 |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "records": [
      {
        "id": 1,
        "inviteCode": "SHOP2026ABC123",
        "shopId": 101,
        "shopName": "XX门店(海淀店)",
        "expireTime": "2026-05-17 23:59:59",
        "status": "unused",
        "createdAt": "2026-04-17 10:00:00"
      },
      {
        "id": 2,
        "inviteCode": "SHOP2026DEF456",
        "shopId": 102,
        "shopName": "XX门店(西单店)",
        "expireTime": "2026-04-18 23:59:59",
        "status": "used",
        "createdAt": "2026-04-16 14:30:00",
```

**路径参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:----:|------|
| id | Long | ✅ | 邀请码记录ID |

**返回参数：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 状态码 |
| message | String | 提示信息 |
| data | null | 无数据 |

**成功响应示例：**

```json
{
  "code": 200,
  "message": "邀请码已作废",
  "data": null
}
```

```json
{

  "message": "该邀请码已被使用，无法作废",
  "data": null
}
```

| 角色编码 | 角色名称 | 说明 |
|----------|----------|------|
| super_admin | 平台管理员 | 系统管理员，可管理所有租户 |
